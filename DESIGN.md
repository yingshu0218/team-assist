# DESIGN.md

## 气质与意象

打开团队管理助手，不是面对冰冷的数字表格，而是翻阅一本精致的财务手账——米白纸面上墨绿色数字沉稳排列，收支用色彩语言区分，间距留白让数据呼吸。专业、克制、值得信赖。每个账本独立呈现，切换时标题与数据自然衔接。

暗色模式下则像深夜书房——深墨蓝底色上翡翠绿的数字依然沉稳，暖光般的点缀让视觉舒适不刺眼。

## 配色方案

### 亮色主题
- 主色：翡翠绿 `#059669` — 财富与生长的意象，沉稳不浮躁
- 收入：翠绿 `#16A34A`
- 支出：朱红 `#DC2626`
- 侧边栏背景：深石板灰 `#1E293B` — 专业、聚焦
- 内容区背景：暖白 `#FAFAF9`
- 卡片背景：纯白 `#FFFFFF`
- 文字主色：近黑 `#18181B`
- 次要文字：石灰灰 `#71717A`

### 暗色主题
- 主色：翡翠绿（稍提亮）`#22C55E`
- 收入：亮翠绿
- 支出：亮朱红
- 侧边栏背景：深墨蓝 `#0F172A`
- 内容区背景：深石墨 `#1C1C1E`
- 卡片背景：深灰 `#2C2C2E`
- 文字主色：近白 `#EDEDEF`
- 次要文字：银灰 `#98989D`

### 语义色（两套主题共用 CSS 变量）
- `--primary` / `--primary-foreground`：主色系
- `--income` / `--expense`：收支语义色
- `--sidebar-*`：侧边栏专用色组

## 字体排版

- 字体族：PingFang SC / Hiragino Sans GB / Microsoft YaHei 系统字体栈
- 数字使用等宽字体变体（tabular-nums）保证金额对齐
- 标题层级清晰：页面标题 1.5rem / 卡片标题 1rem / 正文 0.875rem

## 动效与交互

- 缓动曲线：ease-out 系列，过渡时间 150-200ms
- 卡片悬停轻微上浮 + 阴影增强
- 侧边栏账本切换时内容区淡入过渡
- 图表数据加载时骨架屏占位
- 主题切换无过渡闪烁（disableTransitionOnChange）

## 设计禁忌

- 禁止使用纯蓝/紫渐变
- 禁止过多装饰性插图或 emoji
- 禁止扁平到缺乏层次的设计（每个区块要有呼吸感）
- 禁止在数字展示中使用无衬线粗体以外的字体
