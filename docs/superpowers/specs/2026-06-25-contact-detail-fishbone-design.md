# 联系人详情跟进记录鱼骨视图

## 目标

在 CRM 联系人详情的“跟进记录 → 查看全部”弹窗中，提供与全局联系人时间线一致的列表/鱼骨视图切换，让用户按时间快速浏览单个联系人的跟进轨迹。

## 交互与呈现

- 默认显示现有列表视图；标题区域始终显示“列表视图 / 鱼骨视图”切换。
- 鱼骨视图按日期聚合：按月模式以 `MM.DD` 为节点，按年模式以 `N 月` 为节点。
- 一个节点展示该时间桶内第一条记录摘要；多条记录显示“另有 N 条记录”。
- 鱼骨节点上下交替放置；内容超出弹窗宽度时在鱼骨区域横向滚动，避免压缩卡片内容。
- 全局时间线保留搜索框和相同行为；联系人详情不显示搜索框。

## 技术设计

- 继续使用 `CrmTimelineView`，不新增 API、类型字段或数据请求。
- 将鱼骨分组和显示策略抽为纯函数，以便覆盖日期聚合和年视图分组。
- `CrmTimelineView` 的视图切换不再受 `contactId` 限制；仅搜索框仍仅在全局视图显示。

## 验证

- 单元测试验证同一天记录在按月鱼骨中聚合，及不同年份下按年视图的月份聚合。
- 运行项目 TypeScript 检查和静默 lint；如依赖环境无法执行，明确记录原因。
